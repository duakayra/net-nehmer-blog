Component configuration
====

Newsticker/Blog component can be customized to users' needs by setting the following options via 'Configuration' menu.

##Blog settings##

* __List__ -  this many items on index page
* __Categories__
* __Enable scheduled publishing__
* __Language__

##Schema settings##

* __Database schema__ -  to use for articles
* __Enable commenting to news items__ - requires the [net.nehmer.comments](../../net.nehmer.comments) component
* __Enable AJAX__ -  on-site editing
* __Enable archive browsing__
* __Archive item order__
* __Enable creating news items from remote RSS and Atom feeds__
* __Link items to external URL if available__

##Navigation options

* __Show latest items in navigation__
* __Show pseudo leaves in navigation__
* __Show available feeds in navigation__
* __Use 'view' prefix in url__

##RSS export

* __Enable RSS export__ - activates the URLs rss.xml and rss/$count.xml
* __List this many items in rss.xml__
* __RSS Feed title__ - leave empty to use the topic's title
* __Description__
* __Webmaster email address__
* __Language__
